# Baalvion Jobs Portal (V1)

Baalvion Jobs Portal is a next-generation job marketplace and project collaboration platform designed to connect talent with real-world paid projects and hiring opportunities.

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Firebase (Planned)
- Node.js APIs

## Current Version
V1 – UI and platform structure

## Features (V1)
- Public job marketplace
- Admin dashboard UI
- Project listing system
- Talent discovery pages
- Responsive UI (mobile/tablet/desktop)

## Folder Structure
src/
 ├ app/
 ├ components/
 ├ lib/
 ├ styles/

## Installation

1. Clone repository
git clone https://github.com/baalvionservice/baalvion-jobs-portal.git

2. Install dependencies
npm install

3. Run project
npm run dev

Open http://localhost:3000
